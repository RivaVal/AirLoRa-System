ДАВАЙ УТОЧНИМ ЗАДАЧУ!
На первом этапе задача была:
1. Сформировать проект на котором было бы возможно запустить в Docker Контейнер, который бы эмулировал работу микроконтроллера Raspderry pi Zero 2W, для построения на этом контроллере прием, обработку, сжатие и отправку по сети VIDEO потока в максимально возможном качестве, с доп условиями;
2. Обеспечить взаимодействие   Raspderry pi Zero 2W и ESP32S3 на котором будет работать полетный контроллере, связь будет работать по шине UART, лучше на аппаратном уровне;
3.  Внешнюю связь для проекта будет обеспечивать  Raspderry pi Zero 2W + HAT 4G;
Для решения этой задачи и планировалось создать выше означенную конфигурацию!
Необходимо  разработать код проекта работы с видео потоком на Raspderry pi Zero 2W, для решения этой задачи и задумывались применение Dev Conteiner;
